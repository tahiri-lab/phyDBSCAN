# About the project

Tree classification with dbscan algorithm and multiple dimension, until 250 dimensions.

We use distances between points, not coordinates as usual.

# Installation

Use Makefile to install the project

```
make
```

# Examples

To run the project

```
make run
```

To clean the project

```
make clean
```
